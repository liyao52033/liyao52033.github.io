料箱车与全向车测试原创

# 一 、料箱车

# 1货架结构示意图

# 2 车辆摆放位置

将小车停在原点位置，开机，自动测试前小车必须在原点位置

# 3 软件连接小车

连接GAS-CSC-APP网络，密码12345678，修改IP如下

打开远程监控系统，IP选择172.16.89.102，端口8000，点击打开即可连接到小车

# 4 自动测试

点击软件上方“电机参数设置和监控”，点击自动测试

# 5 问题处理

# 5.1 掉网(已升级程序)

现象是软件上连接已掉线，但小车仍在运行，等小车回到原点时重启小车，然后再重新连接

# 5.2 货叉故障

货叉没有伸出或者伸出没有收回，故障码一般为108，109

处理办法：点击清除报警，货叉归位，完成后再参照故障后小车回原点将小车运行到原点

# 5.3 行走故障

小车行驶过程中报错

处理办法：点击清除报警，完成后再参照故障后小车回原点将小车运行到原点

# 5.4 故障后小车回原点

出现问题后小车会停止自动测试，解决问题后需手动将车移动到原点再自动测试，往原点方向是后退，往换向点是前进；前后走是子道，换向点左右走是母道。

  1. 在原点处停止

    直接点击自动测试即可

  2. 在2巷道处停止

    点击慢速后退到原点附近，然后点击慢速后退找点到原点，点击自动测试

  3. 在2巷道换向点处停止

    看小车是不是在母道(顶升状态)，如果是点击切换到子道，将当前坐标设置为（4368，11320），将目标跑点测试坐标设置为原点坐标（4368，1000），点击运行到原点位置，点击自动测试

  4. 在1巷道换向点处停止

    看是否在子道(非顶升状态)，如果是切换到母道，将当前坐标设置为（1000，11320），将目标跑点测试坐标设置为（4368，11320），勾选换向点，然后点击切换到子道，将目标跑点测试坐标设置原点坐标（4368，1000），点击运行到原点位置，点击自动测试

  5. 在1巷道处停止

    点击慢速后退到1巷道最前方附近，点击慢速后退找点，将当前坐标设置为（1000，1000），将目标跑点测试坐标设置为（1000，11320），勾选换向点，点击运行1巷道换向点处，然后参照在1巷道换向点处停止

# 二、 全向车

# 1 货架结构示意图

# 2 车辆摆放位置

将小车停在原点位置，开机，自动测试前小车必须在原点位置

# 3 软件连接小车

连接GAS-CSC-APP网络，密码12345678，修改IP为172.16.89.102，打开远程监控系统，IP选择172.16.89.102，端口8001，点击打开即可连接到小车

# 4 自动测试

点击软件上方“电机参数设置和监控”，点击自动测试

# 5 问题处理

出现问题后小车会停止自动测试，解决问题后需手动将车移动到原点再自动测试。往原点方向是后退，往充电点方向是前进

点击清除报警，货叉归位(货叉故障时)，慢速前进(后退)走到最近换向点附近，前进(后退)找点到换向点，设置当前坐标与目标跑点测试坐标回到原点点击自动测试
