tsconfig.jsonиҜҰи§ЈдёҺеёёз”Ёй…ҚзҪ®еҺҹеҲӣ

# жҰӮиҝ°

еҸӮиҖғж–ҮжЎЈпјҡtsconfig.json В· TypeScriptдёӯж–ҮзҪ‘ В· TypeScriptвҖ”вҖ”JavaScriptзҡ„и¶…йӣҶ (tslang.cn) (opens new window)

еҰӮжһңдёҖдёӘзӣ®еҪ•дёӢеӯҳеңЁдёҖдёӘtsconfig.jsonж–Үд»¶пјҢйӮЈд№Ҳе®ғж„Ҹе‘ізқҖиҝҷдёӘзӣ®еҪ•жҳҜTypeScriptйЎ№зӣ®зҡ„ж №зӣ®еҪ•гҖӮ tsconfig.jsonж–Үд»¶дёӯжҢҮе®ҡдәҶз”ЁжқҘзј–иҜ‘иҝҷдёӘйЎ№зӣ®зҡ„ж №ж–Үд»¶е’Ңзј–иҜ‘йҖүйЎ№гҖӮ tsconfig.jsonж–Үд»¶еҸҜд»ҘжҳҜдёӘз©әж–Үд»¶пјҢйӮЈд№ҲжүҖжңүй»ҳи®Өзҡ„ж–Үд»¶йғҪдјҡд»Ҙй»ҳи®Өй…ҚзҪ®йҖүйЎ№зј–иҜ‘гҖӮ еңЁе‘Ҫд»ӨиЎҢдёҠжҢҮе®ҡзҡ„зј–иҜ‘йҖүйЎ№дјҡиҰҶзӣ–еңЁtsconfig.jsonж–Үд»¶йҮҢзҡ„зӣёеә”йҖүйЎ№ дёҖдёӘйЎ№зӣ®еҸҜд»ҘйҖҡиҝҮд»ҘдёӢж–№ејҸд№ӢдёҖжқҘзј–иҜ‘пјҡ

  • дёҚеёҰд»»дҪ•иҫ“е…Ҙж–Үд»¶зҡ„жғ…еҶөдёӢи°ғз”ЁtscпјҢзј–иҜ‘еҷЁдјҡд»ҺеҪ“еүҚзӣ®еҪ•ејҖе§ӢеҺ»жҹҘжүҫtsconfig.jsonж–Үд»¶пјҢйҖҗзә§еҗ‘дёҠжҗңзҙўзҲ¶зӣ®еҪ•гҖӮ
  • дёҚеёҰд»»дҪ•иҫ“е…Ҙж–Үд»¶зҡ„жғ…еҶөдёӢи°ғз”ЁtscпјҢдё”дҪҝз”Ёе‘Ҫд»ӨиЎҢеҸӮж•°--projectпјҲжҲ–-pпјүжҢҮе®ҡдёҖдёӘеҢ…еҗ«tsconfig.jsonж–Үд»¶зҡ„зӣ®еҪ•гҖӮ еҪ“е‘Ҫд»ӨиЎҢдёҠжҢҮе®ҡдәҶиҫ“е…Ҙж–Үд»¶ж—¶пјҢtsconfig.jsonж–Үд»¶дјҡиў«еҝҪз•ҘгҖӮ

й…ҚзҪ®йЎ№ иҜҙжҳҺ
extends з»§жүҝй…ҚзҪ®
compileOnSave и®©IDEеңЁдҝқеӯҳж–Үд»¶зҡ„ж—¶еҖҷж №жҚ®tsconfig.jsonйҮҚж–°з”ҹжҲҗж–Үд»¶ иҰҒжғіж”ҜжҢҒиҝҷдёӘзү№жҖ§йңҖиҰҒVisual Studio 2015пјҢ TypeScript1.8.4д»ҘдёҠе№¶дё”е®үиЈ…atom-typescript (opens new window)жҸ’д»¶гҖӮ
compilerOptions зј–иҜ‘йҖүйЎ№пјҢиҜҰи§ҒcompilerOptions
files жҢҮе®ҡдёҖдёӘеҢ…еҗ«зӣёеҜ№жҲ–з»қеҜ№ж–Үд»¶и·Ҝеҫ„зҡ„еҲ—иЎЁ
include жҢҮе®ҡдёҖдёӘж–Үд»¶globеҢ№й…ҚжЁЎејҸеҲ—иЎЁ
exclude жҢҮе®ҡдёҖдёӘж–Үд»¶globеҢ№й…ҚжЁЎејҸеҲ—иЎЁ
references дёҖдёӘеҜ№иұЎзҡ„ж•°з»„пјҢжҢҮжҳҺиҰҒеј•з”Ёзҡ„е·ҘзЁӢгҖӮ жҜҸдёӘеј•з”Ёзҡ„pathеұһжҖ§йғҪеҸҜд»ҘжҢҮеҗ‘еҲ°еҢ…еҗ«tsconfig.jsonж–Үд»¶зҡ„зӣ®еҪ•пјҢжҲ–иҖ…зӣҙжҺҘжҢҮеҗ‘еҲ°й…ҚзҪ®ж–Үд»¶жң¬иә«пјҲеҗҚеӯ—жҳҜд»»ж„Ҹзҡ„пјү

# files include exclude ж–Үд»¶еҢ…еҗ«

"files"жҢҮе®ҡдёҖдёӘеҢ…еҗ«зӣёеҜ№жҲ–з»қеҜ№ж–Үд»¶и·Ҝеҫ„зҡ„еҲ—иЎЁгҖӮ "include"е’Ң"exclude"еұһжҖ§жҢҮе®ҡдёҖдёӘж–Үд»¶globеҢ№й…ҚжЁЎејҸеҲ—иЎЁгҖӮ ж”ҜжҢҒзҡ„globйҖҡй…Қз¬Ұжңүпјҡ

  • * еҢ№й…Қ0жҲ–еӨҡдёӘеӯ—з¬ҰпјҲдёҚеҢ…жӢ¬зӣ®еҪ•еҲҶйҡ”з¬Ұпјү
  • ? еҢ№й…ҚдёҖдёӘд»»ж„Ҹеӯ—з¬ҰпјҲдёҚеҢ…жӢ¬зӣ®еҪ•еҲҶйҡ”з¬Ұпјү
  • **/ йҖ’еҪ’еҢ№й…Қд»»ж„Ҹеӯҗзӣ®еҪ•
й…ҚзҪ® иҜҙжҳҺ
glob жЁЎејҸйҮҢзҡ„жҹҗйғЁеҲҶеҸӘеҢ…еҗ«*жҲ–иҖ….* д»…жңүж”ҜжҢҒзҡ„ж–Үд»¶жү©еұ•еҗҚиў«еҢ…еҗ«еңЁеҶ…пјҲ.ts, .tsx, .d.tsпјү allowJs = true ж—¶пјҢиҝҳеҢ…еҗ« .js е’Ң .jsx
дёҚжҢҮе®ҡ"files"е’Ң"include" зј–иҜ‘еҷЁй»ҳи®ӨеҢ…еҗ«еҪ“еүҚзӣ®еҪ•е’Ңеӯҗзӣ®еҪ•дёӢжүҖжңүзҡ„TypeScriptж–Үд»¶пјҲ.ts, .d.ts е’Ң .tsxпјүпјҢжҺ’йҷӨеңЁ"exclude"йҮҢжҢҮе®ҡзҡ„ж–Үд»¶гҖӮ allowJs = true ж—¶пјҢиҝҳеҢ…еҗ« .js е’Ң .jsx
жҢҮе®ҡ"files"жҲ–"include" зј–иҜ‘еҷЁдјҡе°Ҷе®ғд»¬з»“еҗҲдёҖе№¶еҢ…еҗ«иҝӣжқҘ дҪҝз”Ё"include"еј•е…Ҙзҡ„ж–Үд»¶еҸҜд»ҘдҪҝз”Ё"exclude"еұһжҖ§иҝҮж»Ө з„¶иҖҢпјҢйҖҡиҝҮ "files"еұһжҖ§жҳҺзЎ®жҢҮе®ҡзҡ„ж–Үд»¶еҚҙжҖ»жҳҜдјҡиў«еҢ…еҗ«еңЁеҶ…пјҢдёҚз®Ў"exclude"еҰӮдҪ•и®ҫзҪ® д»»дҪ•иў«"files"жҲ–"include"жҢҮе®ҡзҡ„ж–Үд»¶жүҖеј•з”Ёзҡ„ж–Үд»¶д№ҹдјҡиў«еҢ…еҗ«иҝӣжқҘгҖӮ A.tsеј•з”ЁдәҶB.tsпјҢеӣ жӯӨB.tsдёҚиғҪиў«жҺ’йҷӨпјҢйҷӨйқһеј•з”Ёе®ғзҡ„A.tsеңЁ"exclude"еҲ—иЎЁдёӯгҖӮ
"exclude" й»ҳи®Өжғ…еҶөдёӢдјҡжҺ’йҷӨnode_modulesпјҢbower_componentsпјҢjspm_packagesе’Ң<outDir>зӣ®еҪ•
еңЁcompilerOptionsдёӯжҢҮе®ҡзҡ„"outDir" ж°ёиҝңдјҡиў«зј–иҜ‘еҷЁжҺ’йҷӨпјҢйҷӨйқһдҪ жҳҺзЎ®ең°дҪҝз”Ё"files"е°Ҷе…¶еҢ…еҗ«иҝӣжқҘпјҲиҝҷж—¶е°ұз®—з”ЁexcludeжҢҮе®ҡд№ҹжІЎз”Ёпјү
зү№ж®Ҡжғ…еҶө зј–иҜ‘еҷЁдёҚдјҡеҺ»еј•е…ҘйӮЈдәӣеҸҜиғҪеҒҡдёәиҫ“еҮәзҡ„ж–Үд»¶пјӣжҜ”еҰӮпјҢеҒҮи®ҫжҲ‘д»¬еҢ…еҗ«дәҶindex.tsпјҢйӮЈд№Ҳindex.d.tsе’Ңindex.jsдјҡиў«жҺ’йҷӨеңЁеӨ–гҖӮ йҖҡеёёжқҘи®ІпјҢдёҚжҺЁиҚҗеҸӘжңүжү©еұ•еҗҚзҡ„дёҚеҗҢжқҘеҢәеҲҶеҗҢзӣ®еҪ•дёӢзҡ„ж–Үд»¶
